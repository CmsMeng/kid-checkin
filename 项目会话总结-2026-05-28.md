## 2. 核心功能

- **作业管理**: 添加/删除作业，科目分类（语文/数学/英语/科学/其他）
- **计时器**: 绝对时间戳计时，后台/锁屏不丢失
- **智能预估**: 按字数自动预估用时，可手动 ±1 分钟调整
- **休息提醒**: 完成一项后强制休息 5 分钟，科目全部完成额外奖励 5 分钟
- **闹钟系统**: 休息结束弹出闹钟弹窗，持续响铃，需手动点击「确定」关闭
- **完成报告**: Chart.js 饼图 + 柱状图，含用时统计、科目占比、按时完成率
- **彩纸庆祝**: 全部完成时撒花 + 报告
- **数据持久化**: `localStorage` 键 `hw_checkin_v2`，按天自动重置

### 4.2 休息计时器修复（锁屏/后台闹钟）

**问题**: 锁屏或切换 App 后 `setInterval` 被节流，闹钟不响

**修改**:

| 改动 | 说明 |
|---|---|
| 新增 `_restEndAt` 字段 | 记录休息结束的绝对时间戳 |
| `tickRest()` 重写 | 用 `ceil((_restEndAt - Date.now()) / 1000)` 计算剩余秒数 |
| `_catchUpTimers()` 增强 | `visibilitychange`/`focus` 事件触发时立即同步，用 `_restEndAt` 判断 |
| Notification API | 首次交互请求通知权限，后台到期发系统通知 |
| 清理路径 | `endRest()` / `skipRest()` / `startNewRound()` 均清理 `_restEndAt` |

### 4.3 闹钟系统（之前已完成）

- `showAlarmOverlay()`: 弹出闹钟弹窗 + 每 1.8s 循环响铃
- `dismissAlarm()`: 点击「确定」关闭闹钟
- 锁屏后 `window.focus` + `visibilitychange` 双事件补触发
- `_pendingBeep` + `_onUserInteraction` 补播机制（AudioContext 被挂起时）

## 5. 代码架构速查

### AppState 结构

```js
AppState = {
  date, tasks[], activeTaskId,
  timerIntervalId, timerStartAt,
  isResting, restSeconds, restTotal, restIntervalId,
  _restStartedAt, _restEndAt, restSubjectLabel, totalRestSeconds,
  estimateMinutes(), totalEstimate(), totalActualMinutes(),
  save(), load()
}
```

### 关键函数

| 函数 | 用途 |
|---|---|
| `Actions.startTask(id)` | 开始作业计时 |
| `Actions.finishTask(id)` | 完成作业 → 触发休息 |
| `Actions.startRest(sec, label, elapsed)` | 开始休息倒计时 |
| `tickRest()` | 每秒更新休息倒计时，到期调 `endRest()` |
| `endRest()` | 结束休息 → 播放提示音 → 弹闹钟 → 发通知 |
| `_catchUpTimers()` | 页面恢复可见时同步所有计时器 |
| `_onUserInteraction()` | 恢复 AudioContext + 补播 beep + 请求通知权限 |
| `playBeep()` | 3 段闹钟提示音（880/1100/1320Hz） |
| `showAlarmOverlay()` | 弹出闹钟弹窗 + 循环响铃 |
| `dismissAlarm()` | 关闭闹钟 |
| `showReport()` | 展示 Chart.js 学习报告 |
| `fireConfetti()` | 撒花庆祝 |

### 音频系统

```
_ensureAudioCtx() → 创建/恢复共享 AudioContext
_scheduleBeep(ctx)  → 3音闹钟 (880→1100→1320Hz)
_scheduleDoneBeep(ctx) → 2音完成提示 (660→880Hz)
playBeep()          → 对外接口，处理挂起+补播
```

### 计时器同步链

```
setInterval(tickRest, 1000)     → 正常每秒更新
visibilitychange / window.focus → _catchUpTimers() → tickRest()
用户交互(touch/click)            → _onUserInteraction() → 恢复AudioContext + 补播
```
