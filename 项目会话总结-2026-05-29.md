## 4. 技术要点备忘

- 百度 OCR API 不支持浏览器直接调用（无 CORS 头），必须通过后端代理
- 代理层代码约 20 行 JavaScript，部署到 Cloudflare Workers 即可
- 前端改动：将 Tesseract.js 替换为调用代理 → 百度 API 的 fetch 请求
- 图片预处理可保留，有助于提高云端识别率

### 新 OCR 流程

```
浏览器拍照 → 图片预处理(Canvas) → fetch → Cloudflare Worker → 百度 OCR API → 返回文本 → 智能解析 → 可编辑结果
```

### Worker 部署步骤（待执行）

```bash
cd ocr-proxy
npm install -g wrangler        # 安装 Wrangler CLI
npx wrangler login              # 登录 Cloudflare
npx wrangler secret put BAIDU_API_KEY     # 设置百度 API Key
npx wrangler secret put BAIDU_SECRET_KEY  # 设置百度 Secret Key
npx wrangler deploy             # 部署
# 记下 Worker URL → 填入 index.html 的 OCR_WORKER_URL
```

### 百度 AI 平台注册

1. 访问 https://ai.baidu.com/ → 注册/登录
2. 控制台 → 文字识别 → 创建应用
3. 获取 API Key 和 Secret Key
4. 免费额度：500 次/天（通用文字识别标准版）
